# Context Navigation

> Point d'entrée de la base de connaissances du projet SYNAPPSE.

## Structure

| Dossier      | Contenu                                        |
| ------------ | ---------------------------------------------- |
| `project/`   | Vision, périmètre, planning, parties prenantes |
| `domain/`    | Domaine métier : HaloPSA, données, RGPD        |
| `technical/` | Stack technique, architecture, contraintes     |

## Charger le contexte selon la tâche

| Tâche                        | Fichiers à charger                                 |
| ---------------------------- | -------------------------------------------------- |
| Comprendre le projet         | `project/overview.md`                              |
| Travailler sur le ML         | `domain/halopsa.md` + `technical/stack.md`         |
| Travailler sur le backend    | `technical/stack.md` + `technical/architecture.md` |
| Travailler sur la conformité | `domain/rgpd.md`                                   |
| Planifier un sprint          | `project/overview.md` + `project/planning.md`      |
